## PROBLEM
- 월세 임대차 계약에서 발생하는 계약 위반 분쟁을 막고자 합니다.
- 대학가 자취방에서 자주 발생하는 보증금 분쟁을 막고자 합니다.

## SOLUTION
- <부동산임대차계약서>와 보증금을 블록체인화 하여 투명하고, 안전한 거래를 추구합니다.
- Solidity를 통해서 '이더리움 코인'으로 이루어진 보증금의 거래가 이루어집니다.
